BMAPjs

A BOB! transaction parser that makes many BSV OP_RETURN protocols easier to work with. BMAPjs knows about each supported protocol and will transform BOB transactions into readable, easy to understand objects.

Supported protocols: B, MAP, BAP, MetaNet, Bitcom, AIP, HAIP, SymRe, Bitkey, RON, and BitPic. All other protocol prefixes will retain their original BOB format.

The parser, which uses the Open BSV license, is available on github.

Github

BMAPjs is used by the BMAP-Planaria which provides a simple search query syntax: Twetch, MetaLens, BitKey, BitPic >, Hona >, Jamify
